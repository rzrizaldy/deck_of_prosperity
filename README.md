## 🏙️ What is This Madness?

**Deck of Capitalist** is a roguelike deckbuilder where you're not just playing cards—you're building an empire. Think **Balatro** meets **Monopoly Deal** in a neon-soaked, cyberpunk Jakarta where property is power and greed is good.

Draft Indonesian properties. Form monopolies. Hire corrupt Tycoons. Watch your wealth pile up like a crooked developer's land bank.

### 🎯 The Pitch

- **🃏 Strategic Deckbuilding**: Every card is a piece of Jakarta's real estate puzzle
- **🎰 Poker-Inspired Scoring**: Form monopolies, joint ventures, and conglomerates
- **💼 Hire Tycoons**: Passive buffs that would make any oligarch jealous
- **📈 Escalating Difficulty**: Each round demands more rent. Can you keep up?
- **🌃 Cyberpunk Aesthetic**: Neon lights, dark cards, and that sweet, sweet capitalism glow
- **📱 Mobile-First**: Play on your phone while pretending to work
- **💾 Auto-Save**: Your empire persists. No excuses.

## 🎮 How to Play

### The Core Loop

1. **📋 DRAFT**: Get dealt 8 cards from your deck
2. **🎯 SELECT**: Pick up to 5 cards to play
3. **💰 SCORE**: Form hands (Monopolies, Pairs, etc.) to earn rent
4. **🛍️ SHOP**: Spend your dirty money on Tycoons (permanent buffs)
5. **🔄 REPEAT**: Until you hit the rent target... or go bankrupt

### 🏆 Hand Rankings (Best to Worst)

| Hand | Description | Base Multiplier |
|------|-------------|-----------------|
| **🚂 Transport Network** | 4 Railroads | 15x |
| **🌈 Diversified Portfolio** | 5 different colors | 12x |
| **🏢 Conglomerate** | Full monopoly + partial set | 10x |
| **🎨 Monopoly** | Complete color set | 8x |
| **🤝 Joint Venture** | Two pairs | 6x |
| **🏗️ Development** | One pair | 3x |
| **📉 Liquidation** | Highest single card | 1x |

### 🎴 Card Types

**🏘️ Properties**: 9 color groups from luxury (SKY) to slums (BROWN)
- Each color has different values and set sizes
- Complete sets = Monopolies = BIG MONEY

**🚂 Railroads**: The transport backbone
- 4 unique stations across Jakarta
- Collect all 4 for massive multipliers

**⚡ Utilities**: Power and water (because even oligarchs need electricity)
- 2 utility companies
- Pair them for bonus points

### 💼 Tycoons (Passive Buffs)

Hire these morally questionable characters to boost your empire:

- **Property Mogul**: +20% to all property values
- **Railroad Baron**: +50% to railroad values
- **Utility King**: +100% to utility values
- **Monopolist**: +30% when you score a monopoly
- **Venture Capitalist**: +25% to joint ventures
- **Risk Taker**: +1 extra discard per round
- **Card Shark**: Draw +1 extra card
- **Efficiency Expert**: -10% to rent targets

## 🛠️ Run It Yourself

```bash
# Clone this capitalist dream
git clone https://github.com/rzrizaldy/deck_of_capitalist.git
cd deck_of_capitalist

# Install dependencies (the only ethical thing here)
npm install

# Start the dev server
npm run dev

# Build for production (when you're ready to IPO)
npm run build
```

Then open `http://localhost:5173` and start your journey to becoming Jakarta's most ruthless property tycoon.
